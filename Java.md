# Java

## 内存

### 堆

：存放new的对象和数组；

：可以被所有线程共享，不会存放别的对象引用；

### 栈

：存放基本变量类型

：引用对象的变量

### 方法区(特殊的堆)

：可以被所有的线程共享

：包含所有class和static变量



## 类的加载过程

1. Load：类的加载，将class文件读入内存，并将这些静态数据转换为方法区运行时的数据结构，然后创建一个java.lang.Class对象；

2. Link：类的链接，将类的二进制数据合并到JVM的运行环境中；

3. Initializa：类的初始化
